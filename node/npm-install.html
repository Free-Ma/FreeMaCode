解决 #

原因是 npm 版本不对应

使用 nvm 指定 node 和 npm 版本;或者使用 yarn 作为包管理器

TIP

删除 package-lock.json 文件

如果有 node_modules 文件夹，也要删除

重新使用 yarn 下载 package.json 中的依赖包
